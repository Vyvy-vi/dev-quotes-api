Dev-Quotes-API

status: OFFLINE License: BSD Version: v0.1 docs: #todo

The dev-quotes-api is a REST API made using Flask, and HarperDb for showcasing community quotes from developers and possibly in the future this might be connected to different services, Just for fun :P The API is currently in development and might be launched soon. This is an opensource project under the "BSD 3-Clause" License.

NOTE- This is a synchronous API

  • Usage of the API

    - Fetching random quotes from the API
    GET   /quote/?author={name}&num={x} 
            Fetches a random quote from the api. Click on GET button above for response example. 
        Optional arguments: num- number of quotes to fetch
                            author- fetch quotes by a particular author
        Response Codes: 200 - OK
                        404 - Resource not Found
                        400 - Bad Request, when author isn't found or num argument is too high
                        505 - Internal server Error (Kindly report these by opening an issue on the github repo)
    GET   /quote/random/?num={x} 
            Fetches random quotes from the API. This is different from the prior query, since this is faster.
        Click on the GET button above for a response example.
        Optional arguments: num- number of quotes to fetch(defaults to 1).
                            If value of num is too high, this query returns all the results in the db.
        Response Codes: 200 - OK
                        400 - Bad Request, when num argument is too high
                        505 - Internal server Error (Kindly report these by opening an issue on the github repo)
    - Fetching quotes by id and author-id
    GET   /quote/{id}
            Fetches random quote from the API with the id = {id}. The ids are purely relative
        to when a quote was entered in the db.
        This one is mainly for making references to a particular quote on the db.
        Click on the GET button above for a response example.
        Required arguments: id- id of the quote to fetch.
        Response Codes: 200 - OK
                        400 - Bad Request, when id isn't found
                        505 - Internal server Error (Kindly report these by opening an issue on the github repo)
    GET   /quote/author/{id}
            Fetches all quotes from the API by the author with id = {id}. The ids are purely relative
        to when an author was entered in the db.
        This one is mainly for making references to a particular author on the db.
        Click on the GET button above for a response example.
        Required arguments: id- id of the author whose quotes to fetch
        Response Codes: 200 - OK
                        400 - Bad Request, when author with this id isn't found
                        505 - Internal server Error (Kindly report these by opening an issue on the github repo)
    - Fetching quotes by author name
    GET   /quote/author/{name}
            Fetches all quotes from the API with the author as the queried name.
        Click on the GET button above for a response example.
        Required arguments: name- name of the author to fetch.
        Response Codes: 200 - OK
                        404 - Not Found, when author isn't found
                        505 - Internal server Error (Kindly report these by opening an issue on the github repo)
